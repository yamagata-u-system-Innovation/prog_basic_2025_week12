# 第12週"CSVデータをグラフで可視化し、平均による変化を観察する"


## この課題について

この課題は、プログラミング初心者向けに  
**「データをグラフで見て考える」** ことを目的としています。

株価のダミーデータ（実在の投資とは無関係）を題材に、

- CSVデータを読み込む
- 数値の変化をグラフで確認する
- 平均を使うと見え方がどう変わるかを体験する

ことを行います。

※ 投資の正解・不正解を考える課題ではありません。

---

## 学習の流れ（今回扱う範囲）

### Step0：データの確認（完成済み）
- CSVが正しく読み込めているか確認
- データ件数、最小値・最大値を表示

### Step1：株価データをグラフ表示する（必須）
- 株価（終値）を折れ線グラフで表示
- 数字の変化を目で確認する

### Step2：移動平均を追加する（今回の到達点）
- 株価のグラフに **移動平均（5日）** を追加
- 「そのままの値」と「平均した値」の違いを観察する

※ Step3（買いタイミングの判定）は **今回の講義では扱いません**。

---

## Step2 のゴール（今回の到達点）

Step2 では、株価のグラフに **移動平均（5日）** を追加します。

- 株価（終値）：ギザギザした線  
- 移動平均：なめらかな線  

この 2 本の線を見比べて、

- 株価の全体的な流れ  
- 上がっているか、下がっているか  

が **目で確認できれば成功**です。

※ 移動平均の計算方法を完全に理解する必要はありません。  
※ 「計算すると、見え方が変わる」ことを体験するのが目的です。

---

はじめに、必要なライブラリをインストールします。
